## Voting-application

**Задание:**
````
Есть N (вынести в конфиг) кандидатов. 
Каждые 10мс с равной долей вероятности генерируется голос 
за одного из кандидатов. Голосование длится 9с (конфиг). 
Пользователь видит время до окончания голосования и отсортированную 
таблицу с кандидатами и их результатами голосования в процентах 
(округлить до десятых, учесть тот факт, что сумма всех процентов 
должна быть строго равна 100). Данные обновляются 1 раз в секунду. 
По окончанию выводится сообщение о победителе.
````
**При разработке использовать:**

- ES6/7 как основной язык приложения
- React для пользовательского интерфейса
- Bootstrap
